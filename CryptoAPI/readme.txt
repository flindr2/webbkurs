1. Kör "npm install" för att ladda ner alla npm-paket (lagras i node_modules)
2. Starta MongoDB med kommandprompten/terminalen "mongod.exe" i mappen där MongoDB ligger.
	Windows: C:\Program Files\MongoDB\Server\4.4\bin
	"cd C:\Program Files\MongoDB\Server\4.4\bin" för att komma dit.
3. Kör "npm run start" för att starta APIet.
